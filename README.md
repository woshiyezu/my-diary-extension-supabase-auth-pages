# my-diary-extension-supabase-auth-pages

1. Supabase ダッシュボード
1. 「Auth」> 「Settings」 > 「URL Configuration」
1. 以下を正しく設定：
   | フィールド | 設定内容 |
   | ----------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------- |
   | **Site URL** | `https://woshiyezu.github.io/my-diary-extension-supabase-auth-pages` |
